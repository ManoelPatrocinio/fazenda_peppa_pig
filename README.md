# Portfólio
  Aplicação avaliativa da trilha de Front-End, onde é proposto a criação de um sistema  para controlar a criação de suínos numa fazenda de suinocultura.

[Confira o projeto](https://fazenda-peppa-pig.vercel.app/)



## Tecnologias usadas
  * Angular 17
  * TypeScript
  * Sweetalert 2
  * HTML5 
  * CSS3
  * JS
